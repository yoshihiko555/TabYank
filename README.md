# TabYank

選択したタブのURLをクリップボードにコピーするChrome拡張機能。

## 機能

- ポップアップUIでタブ一覧を表示
- チェックボックスまたは行クリックでタブを選択
- 全選択/全解除ボタンで一括操作
- 選択したタブのURLを改行区切りでクリップボードにコピー

## インストール

1. このリポジトリをクローン
2. ブラウザで `chrome://extensions/` を開く（Diaの場合は `dia://extensions`）
3. 「デベロッパーモード」をONにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. クローンしたフォルダを選択

## 使い方

1. ツールバーのTabYankアイコンをクリック
2. コピーしたいタブにチェックを入れる
3. 「コピー」ボタンをクリック

## 対応ブラウザ

- Google Chrome
- Dia Browser
- その他Chromiumベースのブラウザ
