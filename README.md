# TabYank

選択したタブのURLをクリップボードにコピーするChrome拡張機能。

## 機能

- ポップアップUIでタブ一覧を表示
- 行クリックでタブを選択/解除（チェックボックス不要）
- `Shift + クリック`で範囲選択/解除
- 全選択/全解除ボタンで一括操作
- 選択したタブのURLを改行区切りでクリップボードにコピー

## インストール

1. このリポジトリをクローン
2. ブラウザで `chrome://extensions/` を開く（Diaの場合は `dia://extensions`）
3. 「デベロッパーモード」をONにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. クローンしたフォルダを選択

## 使い方

1. ツールバーのTabYankアイコンをクリック
2. コピーしたいタブの行をクリックして選択する（複数は `Shift + クリック` で範囲選択可能）
3. 「コピー」ボタンをクリック

## 対応ブラウザ

- Google Chrome
- Dia Browser
- その他Chromiumベースのブラウザ
